## 🟡 What is Beginning Blends?

Beginning blends are two or three consonants that appear at the start of a word. Each letter sound is still heard clearly when spoken.

### Simple Examples:
- **bl** → black  
- **br** → brave  
- **cl** → clap  
- **tr** → tree  

These blends help children recognize sound patterns and read new words faster.

## ⚫ How to Use This (Parents/Teachers Guide)

Follow these simple tips to help children learn effectively:

- Start with one worksheet at a time  
- Read the word list together  
- Encourage kids to say sounds aloud  
- Let them try independently first
